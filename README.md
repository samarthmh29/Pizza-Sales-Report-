🍕 Pizza Sales Dashboard - Power BI
📌 Project Overview

This Power BI project analyzes pizza sales data to uncover business insights, sales trends, customer ordering patterns, and product performance. The dashboard provides interactive visualizations and KPI tracking to support data-driven decision-making.

📊 Key Performance Indicators (KPIs)
💰 Total Revenue: 790.01K
🛒 Total Orders: 21K
🍕 Total Pizzas Sold: 48K
📈 Average Order Value: 38.28
📦 Average Pizzas Per Order: 2.32
📈 Dashboard Features
Home Page
Daily Trend for Total Orders
Monthly Trend for Total Orders
Sales Distribution by Pizza Category
Sales Distribution by Pizza Size
Total Pizzas Sold by Category
Dynamic Date and Category Filters
Best/Worst Sellers Page
Top 5 Pizzas by Revenue
Top 5 Pizzas by Quantity
Top 5 Pizzas by Orders
Bottom 5 Pizzas by Revenue
Bottom 5 Pizzas by Quantity
Bottom 5 Pizzas by Orders
